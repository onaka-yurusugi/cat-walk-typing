# SOW: 黒猫バシバシタイピング ゲーム体験大幅強化

**作成日**: 2026-03-14
**ステータス**: 完了

---

## 概要

ゲームの面白さ・リプレイ性・爽快感を大幅に向上させるため、以下6カテゴリの機能を一括実装した。

---

## 1. サウンド＆フィードバック強化

| 項目 | 内容 | ステータス |
|------|------|-----------|
| タイプ音 | 正解時にランダムな猫の鳴き声SE（Web Audio API合成） | ✅ |
| コンボ演出 | コンボ10/20/50/100で画面揺れ＋フローティングテキスト | ✅ |
| ミス演出 | ミス時に赤フラッシュ＋「シャーッ」ノイズSE | ✅ |

### 実装詳細
- `src/hooks/useSound.ts` — Web Audio APIでオシレーター合成。外部音声ファイル不要。
  - `playMeow()`: ランダム周波数のサイン波で「にゃっ」風SE
  - `playHiss()`: ホワイトノイズ＋ハイパスフィルタで「シャーッ」風SE
  - `playPurr()`: 低周波ソートゥース波＋LFOで「ゴロゴロ」風SE

## 2. 難易度バリエーション

| 項目 | 内容 | ステータス |
|------|------|-----------|
| 難易度選択 | 子猫(30s/短め)・成猫(60s/通常)・ボス猫(90s/高難度) | ✅ |
| スピードアップ | 難易度に応じて文字列長・寝落ち率が変動 | ✅ |

### 実装詳細
- `generateCatString.ts` に `Difficulty` 型と `DIFFICULTY_CONFIG` を追加
  - 子猫: 文字列3-6文字, 寝落ち10%, 寝落ち長5-10
  - 成猫: 文字列5-10文字, 寝落ち30%, 寝落ち長15-39（従来と同じ）
  - ボス猫: 文字列8-15文字, 寝落ち50%, 寝落ち長20-50
- タイトル画面に難易度選択ボタンを3つ配置

## 3. ゲーム内イベント

| 項目 | 内容 | ステータス |
|------|------|-----------|
| ゴロゴロタイム | 正解時2%の確率でスコア2倍モード5秒間突入 | ✅ |
| 毛玉ボーナス | コンボ30ごとに+100点ボーナス | ✅ |
| 猫パンチラッシュ | 寝落ちイベント完走時(20文字以上)に画面揺れ＋特別テキスト | ✅ |

### 実装詳細
- ゴロゴロタイム中はスコアUIが金色に光る（`animate-purr-glow`）
- イベント発生時にフローティングテキストが上方向にフェードアウト（`animate-float-up`）

## 4. リザルト画面の充実

| 項目 | 内容 | ステータス |
|------|------|-----------|
| ランク評価 | スコアに応じた7段階の猫の称号表示 | ✅ |
| 統計情報 | 最大コンボ・正確率・打鍵速度・ミス数の表示 | ✅ |
| ハイスコア記録 | localStorageでベストスコア保存＆表示 | ✅ |

### 実装詳細
- `src/lib/ranks.ts` — 7段階ランク: のら猫ウォッチャー → 猫見習い → 猫なで上手 → 猫好き上級者 → 猫使い → 黒猫マスター → 猫神
- `src/lib/highscore.ts` — localStorage CRUD（エラーハンドリング付き）
- リザルト画面にStatCardコンポーネントで4項目グリッド表示
- NEW RECORD時にパルスアニメーション

## 5. ビジュアル強化

| 項目 | 内容 | ステータス |
|------|------|-----------|
| 背景猫シルエット | コンボ10/20/30/50で背景に猫絵文字が1→3→5→8個出現 | ✅ |
| キーボード色変化 | コンボ10で緑、30で紫、50で炎色にテーマ変化 | ✅ |
| 画面揺れ | コンボ50/100＋寝落ち完走時にスクリーンシェイク | ✅ |

### 実装詳細
- `getKeyboardTheme()` でコンボ数に応じたTailwindクラスセットを返す
- `getCatSilhouetteCount()` で背景シルエット数を制御
- CSSキーフレーム `shake`, `silhouette-drift` を追加

## 6. リプレイ性向上

| 項目 | 内容 | ステータス |
|------|------|-----------|
| デイリーチャレンジ | 日付ベースシード値(mulberry32 PRNG)で全員同一問題 | ✅ |
| タイムアタックモード | 10問を最速で打ち切るモード | ✅ |

### 実装詳細
- `generateCatString()` にオプション `seed` パラメータ追加（mulberry32 PRNG）
- `generateDailyChallenge()` で日付ベースのシード値から固定問題セット生成
- タイムアタック/デイリーモードは経過時間表示＋進捗表示（N/10）
- タイトル画面にモード選択ボタン3つ（ノーマル/タイムアタック/デイリー）

## 技術方針（遵守事項）

- 外部ライブラリ追加なし（Web Audio API等ブラウザネイティブ機能のみ）
- TypeScript strict、`any`/`unknown` 不使用
- Tailwind CSS v4のCSS-first設定を継続
- ビルド＆lint全パス確認済み
